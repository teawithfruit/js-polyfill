# js-polyfill
This is a docker container setup based on [Financial-Times/polyfill-service](https://github.com/Financial-Times/polyfill-service).
# Run
## With docker
```
docker run --name js-polyfill -p 8801:8801 -d teawithfruit/js-polyfill:latest
```
## With docker-compose
```
docker-compose up -d
```
